# Guía de Instalación

## Requisitos

- Docker 20.10+
- Docker Compose 2.0+
- Python 3.11+

## Instalación con Docker
```bash
# Clonar repositorio
git clone https://github.com/jefersalcedo/biblioteca_universitaria

# Navegar al directorio
cd biblioteca_universitaria

# Iniciar servicios
docker-compose build --no-cache

docker-compose up 
```

## Acceder

- Frontend: http://localhost:5000
- API Gateway: http://localhost:8000
